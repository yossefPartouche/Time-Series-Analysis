# Time Series Analysis: A Practical & Visual Approach in Python

Welcome to my interactive notebook and project repository for learning **Time Series Analysis**. 

This repository serves as a comprehensive, practical, and highly visual companion to the classic textbook *[Time Series Analysis and Its Applications](https://link.springer.com/book/10.1007/978-3-031-70584-7) by Robert H. Shumway and David S. Stoffer*. 

While the original text relies heavily on R and theoretical derivations, this repository bridges the gap by translating these core statistical concepts into **Python**, emphasizing intuitive visuals and real-world applications.

> **Disclaimer:** This repository is **not** a replacement for the textbook. The book provides significantly more details, rigorous mathematical foundations, and deeper theoretical explanations. Rather, this repository is a supplementary review designed to provide a practical, hands-on implementation perspective specifically geared towards Machine Learning.

## Repository Goals

* **Practical & Visual Understanding:** Stripping back the heavy math to focus on *how* and *why* time series models work, supported by extensive data visualizations.
* **Python Implementations:** Providing modern Python equivalents (using libraries like `pandas`, `statsmodels`, `scipy`, and `scikit-learn`) for the book's foundational concepts.
* **Exercise Solutions:** Step-by-step walkthroughs and code solutions for selected end-of-chapter questions from the textbook.
* **Applied Capstone Projects:** At the end of every chapter, you will find a dedicated mini-project applying that chapter's theory to a real-world dataset to solidify the concepts.

## What's Inside?

The repository is structured chapter-by-chapter. In each chapter's folder, you can expect to find:
1. **`Notes/`**: Jupyter Notebooks containing distilled concepts, visual explanations, and Python code snippets.
2. **`Solutions/`**: My coded solutions to selected exercises from the book.
3. **`Project/`**: The end-of-chapter practical implementation project.
